# supermarket
<br>
需要远程部署联系QQ484139229，加时备注来意
<br>
基于java的超市购物系统，系统包含管理员和客户，包括注册功能。管理员可以添加商品修改商品等信息，客户可以根据商品选择购买商品，有加入购物车功能。IDEA和eclipse均可运行，可远程部署。此系统包含学习文档资料。
